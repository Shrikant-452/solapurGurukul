# Divine Mantra Website

A spiritual website for Hindu mantras with multi-language support.

## Features

- 🌐 Multi-language support (English, Hindi, Marathi, Telugu, Kannada)
- 🔐 User Authentication (Login/Register)
- 👑 Admin Panel (Add/Edit/Delete Mantras)
- 📱 Fully Responsive Design
- 🎨 Beautiful Spiritual UI with Animations
- 🔍 Search and Filter Mantras
- 🛡️ Right-click disabled on images

## Tech Stack

- React 18
- React Router v6
- i18next for translations
- Framer Motion for animations
- CSS3 with custom properties

## Installation

```bash
# Clone the repository
git clone <repo-url>

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
